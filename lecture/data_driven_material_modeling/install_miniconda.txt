1) Go to
	https://docs.conda.io/en/latest/miniconda.html
	and download the corresponding installer.
	Then install miniconda.
	
2) In Windows go to start and type 
	'Anaconda Prompt (miniconda3)'
	Run it.
	
	In Linux just open a terminal.
	
3) In Anaconda Prompt or terminal type
	conda install pkg
	where pkg stands for the required package.
	Install the following packages:
	- numpy
	- tensorflow
	- matplotlib
	- h5py
	- jupyter
	
4) In Anaconda Prompt or terminal type
	jupyter-notebook
	The jupyter interface will pop up.
	Navigate to downloaded files and open up
	the required notebook.
